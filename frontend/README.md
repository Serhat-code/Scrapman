# Scrapman — Frontend

Interface Next.js 16 (App Router) de Scrapman. Voir le
[README à la racine du projet](../README.md) pour l'installation complète
(scraper Python, worker d'envoi, base Supabase, sécurité, anti-spam).

## Démarrage rapide

```bash
npm install
cp .env.local.example .env.local   # voir le README racine pour le détail des variables
npm run dev
```

## Commandes

```bash
npm run dev      # serveur de développement
npm run build    # build de production
npm run lint      # ESLint
npm test          # tests unitaires (vitest)
```
